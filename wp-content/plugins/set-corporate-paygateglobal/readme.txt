Le plugin **SET Corporate MobilePay** vous permet de recevoir des paiements par Flooz ou Tmoney

== Description ==
Le plugin **SET Corporate MobilePay** vous permet de recevoir des paiements par Flooz de Moov Africa (Togo) ou Tmoney
basé sur les services de paygate global

== Frequently Asked Questions ==

= Le plugin est-il de la société PayGate Global ? =

Non, le plugin **SET Corporate MobilePay** n'est pas de paygate global. Il a été conçu par **SET Corporate** sans corrélation avec PayGate Global. C'est un plugin non officiel même s'il permet le paiement par l'intermédiaire de la plateforme de PayGate Global.

= Le plugin est-il gratuit? =

Le plugin **SET Corporate MobilePay** est gratuit sur 10 jours d'éssais, à compter de la date d'nstallation.
